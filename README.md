# UnitedHealth (unitedhealth)
UnitedHealth is a Fortune 500 company that provides developer APIs and integration capabilities through its developer platform. The UnitedHealth Group interoperability APIs provide access to healthcare data including patient records, claims, provider directories, and formulary information in compliance with CMS interoperability rules. They follow HL7 FHIR standards for health data exchange.
